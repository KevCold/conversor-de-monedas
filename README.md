# Welcome to the first Backend Challenge of the One G6 Program - Alura Challenge

![imagen](https://github.com/KevCold/conversor-de-monedas/assets/160421091/b3ebe38c-1fc2-4ff4-aaa4-70d0c7f13dd8)
